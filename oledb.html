OLED B Click Support

  • By Vaishnav M A
  • June 23, 2019
  • Clicks

This post contains a basic test demonstration for the spi based OLED B click support through greybus, This post assumes that GBSIM is correctly installed and setup in your board.OLED B click carries a 96 x 39px blue monochrome passive matrix OLED display. The display is bright, has a wide viewing angle and low power consumption. To drive the display, OLED B click features an SSD1306 controller. It's built-in functionalities include contrast control, normal or inverse image display, vertical and horizontal scrolling functions and more.

Start GBSIM

Before inserting the click manifest start gbsim(if not already running as service) whether using the start script or by separately modprobe the modules and running gbsim.

sudo startgbsim

Insert the Click Manifest using the CLI

Clone and setup the insclick CLI along with the manifesto tool and create the manifest blobs

git clone https://github.com/vaishnav98/manifesto.git
cd manifesto
sudo sh ./install.sh

Now Connect the Click Board to the corresponding slot and then using the insclick CLI the click can be easily interfaced

sudo insclick oledb p1
